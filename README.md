# Banque

Projet Java - MIAGE L2 S2 - Modélisation Orientée Objet

## Contenu

### TP1 - Classes de base
- `tp1/banque` : Gestion de comptes bancaires (Client, Compte)
- `tp1/date` : Classe Date (utilisée dans TP2 et TP6)

### TP2 - Structures de données et tris
- `tp2/exo1-4` : Permutations, tableaux, listes chaînées
- `tp2/exo5` : TableauDate (utilise la classe Date du TP1)
- `tp2/exo6` : ArrayListeDate, tris (utilise la classe Date du TP1)

### TP6 - Patrimoine et exceptions
- `tp6` : Gestion du patrimoine d'une personne (Personne, Bien, Vehicule, Habitat)
- Gestion d'exceptions personnalisées (ExceptionInseeErronee, ExceptionValeurErronee)

> TP1, TP2 et TP6 sont regroupés car TP2 et TP6 dépendent de la classe `Date` définie dans TP1.
