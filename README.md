# Banque

Modélisation orientée objet d'un système bancaire en Java.

## Contenu

### Gestion bancaire de base
- `tp1/banque/` : Comptes et clients (Client, Compte)
- `tp1/date/` : Classe Date

### Structures de données et tris
- `tp2/exo1-4` : Permutations, tableaux
- `tp2/exo5` : TableauDate
- `tp2/exo6` : ArrayListeDate, algorithmes de tri

### Patrimoine
- `tp6/` : Gestion du patrimoine d'une personne (Personne, Bien, Vehicule, Habitat)
- Exceptions personnalisées (ExceptionInseeErronee, ExceptionValeurErronee)

## Technologie
- Java
